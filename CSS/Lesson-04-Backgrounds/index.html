Topic: CSS Backgrounds

What is a Background?

A background is the area behind an HTML element. CSS provides different properties to change the appearance of backgrounds.

Background Properties

  1. background-color

    Used to set the background color of an element.

  2. background-image

    Used to add an image as the background.

  3. background-repeat

    Controls whether the background image repeats.

  4. background-position

    Sets the starting position of the background image.

  5. background-size

    Defines the size of the background image.

  6. background-attachment

    Defines whether the background image scrolls with the page.
