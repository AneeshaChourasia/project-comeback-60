Total Width Calculation

Formula:
Total Width = Content Width + Left Padding + Right Padding + Left Border + Right Border

Example

       Width : 300px
       Padding : 20px
       Border : 5px solid black
       
       Calculation:
       
       300 + 20 + 20 + 5 + 5 = 350px

Total Height Calculation

Formula:
Total Height = Content Height + Top Padding + Bottom Padding + Top Border + Bottom Border

Example

       Height : 150px
       Padding : 20px
       Border : 5px solid black
       
       Calculation:
       
       150 + 20 + 20 + 5 + 5 = 200px

Important Points

  • Padding is applied on all four sides.
  • For Total Width, only Left and Right padding & border are added.
  • For Total Height, only Top and Bottom padding & border are added.
  • Margin is NOT included in Total Width or Total Height.
  • By default, width and height apply only to the content area.
