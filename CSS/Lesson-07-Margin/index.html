CSS Lesson 07

Topic: Margin

Margin is used to create space outside an element's border. It separates one element from another.

What is Margin?

Margin creates space outside the border of an element.

Margin Properties

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left
  • Margin Shorthand

Example 1 - margin-top

margin-top: 40px;

Example 2 - margin-right

margin-right: 80px;

Example 4 - margin-left

margin-left: 80px;

Margin Shorthand

margin: 20px;
margin: 20px 60px;
margin: 20px 40px 60px;
margin: 20px 40px 60px 80px;

Important Points

  • Margin creates space outside the border.
  • Margin does not affect the content area.
  • Margin can be applied individually or using shorthand.
  • Margin accepts positive and negative values.
  • margin: auto is used to center block elements.

Margin Shorthand Rules

  • margin: 20px; → All sides
  • margin: 20px 40px; → Top-Bottom | Right-Left
  • margin: 20px 40px 60px; → Top | Right-Left | Bottom
  • margin: 20px 40px 60px 80px; → Top | Right | Bottom | Left
