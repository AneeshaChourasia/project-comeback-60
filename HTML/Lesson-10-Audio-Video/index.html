Audio & Video

Controls Attribute

The controls attribute displays the default audio player controls.


Autoplay + Muted Attribute

The autoplay attribute automatically starts playing the audio when the page loads.

The muted attribute starts the audio without sound.


Loop Attribute

The loop attribute plays the audio repeatedly after it ends.


Controls Attribute

The controls attribute displays the default video player controls.


Width Attribute

The width attribute sets the width of the video player.


Height Attribute

The height attribute sets the height of the video player.


Poster Attribute

The poster attribute displays a thumbnail image before the video plays.


Autoplay + Muted Attribute

The autoplay attribute automatically starts playing the video when the page loads.

The muted attribute starts the video without sound.


Loop Attribute

The loop attribute plays the video repeatedly after it ends.
