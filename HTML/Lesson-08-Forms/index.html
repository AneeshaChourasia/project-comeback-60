HTML Lesson 08

Topic: HTML Forms

HTML Forms are used to collect user information such as name, email, password, phone number, gender, feedback, etc.

What is a Form?

The <form> tag is used to create a form that accepts user input.

Basic Form Structure

<form>
    Form Elements
</form>

Label Tag

The <label> tag provides a title for an input field.

The for attribute of the label must match the id of the input.

Rule:

for = id

Common Input Types

  • text
  • password
  • email
  • number
  • tel
  • date
  • radio
  • checkbox
  • file

Important Points

  • <form> is used to create a form.
  • <label> describes an input field.
  • The for attribute must match the input id.
  • The name attribute identifies form data.
  • placeholder displays hint text.
  • Radio buttons allow only one option to be selected.
  • Checkboxes allow multiple options to be selected.
  • Use type="date" to select a date.
  • Use type="file" to upload files.
  • <textarea> is used for multi-line text input.
  • <select> creates a dropdown list.
  • <option> defines items inside a dropdown.
  • button type="submit" submits the form.
  • button type="reset" clears all form fields.
