JavaScript Lesson 05

Topic: Logical Operators

What are Logical Operators?

Logical operators are used to combine two or more conditions. They always return a Boolean value (true or false).

Logical Operators

  1. Logical AND (&&)

    Returns true only if all conditions are true.

    Example:

    10 > 5 && 20 > 15 → true

  2. Logical OR (||)

    Returns true if at least one condition is true.

    Example:

    10 > 20 || 15 > 10 → true

  3. Logical NOT (!)

    Reverses the Boolean value. It changes true to false and false to true.

    Example:

    !(10 > 5) → false

Summary

  • && (Logical AND)
  • || (Logical OR)
  • ! (Logical NOT)
