JavaScript Lesson 04

Topic: Comparison Operators

What are Comparison Operators?

Comparison operators are used to compare two values. They always return a Boolean value:

  • true
  • false

Comparison Operators

  1. Equal To (==)

    Compares only the values.

    Example:

    10 == "10" → true

  2. Strict Equal (===)

    Compares both value and data type.

    Example:

    10 === "10" → false

  3. Not Equal (!=)

    Checks whether two values are different.

    Example:

    10 != 20 → true

  4. Strict Not Equal (!==)

    Checks both value and data type.

    Example:

    10 !== "10" → true

  5. Greater Than (>)

    Checks whether the left value is greater than the right value.

    Example:

    20 > 18 → true

  6. Less Than (<)

    Checks whether the left value is smaller than the right value.

    Example:

    15 < 30 → true

  7. Greater Than or Equal (>=)

    Checks whether the left value is greater than or equal to the right value.

    Example:

    80 >= 80 → true

  8. Less Than or Equal (<=)

    Checks whether the left value is less than or equal to the right value.

    Example:

    30 <= 35 → true

Summary

  • == (Equal To)
  • === (Strict Equal)
  • != (Not Equal)
  • !== (Strict Not Equal)
  • > (Greater Than)
  • < (Less Than)
  • >= (Greater Than or Equal)
  • <= (Less Than or Equal)
